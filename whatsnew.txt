MQTTBox v0.2.2 Release Notes
============================

Release Date: October 3, 2025

🚀 Major Feature Improvements
=============================

1. Fixed MQTT Connection Issues
   - Resolved "net.isIP is not a function" error in Web Workers
   - Redirected MQTT connections to Electron main process to avoid Node.js network modules in renderer process
   - Improved application stability and compatibility

2. Enhanced SSL/TLS Certificate Support
   - Improved AWS IoT Core certificate authentication functionality
   - Support for multiple certificate types: SSC (Server and Client Certificate), CC (Client Certificate), CSSC (Client Self-Signed Certificate)
   - Enhanced certificate file path validation and error handling

3. Improved Electron Integration
   - Optimized IPC communication between main and renderer processes
   - Enhanced MQTT connection state management
   - Improved error handling and reconnection mechanisms

🔧 Technical Improvements
========================

1. Main Process MQTT Service
   - Implemented complete MQTT connection lifecycle management
   - Support for all MQTT operations: connect, disconnect, publish, subscribe
   - Added connection state monitoring and automatic reconnection

2. Build System Optimization
   - Fixed package.json configuration issues
   - Added author email information to support Linux deb package builds
   - Optimized multi-platform build process

3. Platform Compatibility
   - Support for Intel and ARM64 architectures
   - Support for macOS, Windows, and Linux platforms
   - Maintained original Web UI functionality
