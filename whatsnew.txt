MQTTBox v0.2.2 Release Notes
============================

Release Date: October 3, 2025

🚀 Improvements
===============

1. Electron desktop app support
   - Added Electron main process and IPC bridge
   - MQTT handled in main process for stability
   - Eliminated manual Web Worker fixes

2. Build System Updates
   - Multi-platform build optimization
   - Linux AppImage support
   - See Dependency Changes for details

3. Platform Compatibility
   - Intel and ARM64 architectures
   - macOS, Windows, and Linux platforms

🧩 Dependency Changes
=====================

Added
- electron (new): 22.0.0
- electron-builder (new): 24.6.4
- vinyl-buffer (new): 1.0.1

Upgraded
- mqtt: 2.1.3 → 4.3.8
- gulp: 3.9.1 → 4.0.2
